# 演出配置指南

演出系统（Performances）为故事中的关键时刻添加多媒体效果——音效、图片、视频等。当 LLM 回复中出现特定关键词或进入新阶段时，自动触发对应的演出。

## 基本概念

每个演出由以下部分组成：

- **触发条件**（trigger）：什么时候播放
- **渲染器**（renderer）：播放什么类型的内容
- **资源文件**（audio/layers）：具体的音频或图片文件
- **播放策略**（playOnce）：重复播放规则

## 演出类型

| renderer | 说明 | 适用场景 |
|----------|------|---------|
| `audio` | 播放音效 | 技能释放、角色反应 |
| `image` | 显示图片（可附带音效） | 进入新阶段、重要场景 |
| `layeredCss` | 多层动画 | 华丽技能特效 |
| `video` | 播放视频 | 过场动画 |

## 触发类型

### knowledgeUse — 关键词匹配

当 AI 回复文本中包含指定关键词时触发。最常用的触发方式。

```json
{
  "trigger": {
    "type": "knowledgeUse",
    "characterId": "qiaofeng",
    "keywords": ["降龙十八掌", "亢龙有悔", "掌力"],
    "matchBoldOnly": false
  }
}
```

- `characterId`：可选，限定只有该角色的回复才触发
- `keywords`：关键词列表，任一匹配即触发
- `matchBoldOnly`：`true` 时只匹配 `**粗体**` 文本，`false` 匹配全文

### stageEnter — 进入阶段

当剧情推进到指定阶段时触发。适合展示场景图片。

```json
{
  "trigger": {
    "type": "stageEnter",
    "stageId": "stage_3"
  }
}
```

### skillUse — 技能使用

当角色使用指定技能时触发。

```json
{
  "trigger": {
    "type": "skillUse",
    "skillId": "xianglong_kanglongyouhui"
  }
}
```

### firstAppearance — 首次出场

角色第一次发言时触发。

```json
{
  "trigger": {
    "type": "firstAppearance",
    "characterId": "dingchunqiu"
  }
}
```

## 播放策略

| playOnce | 说明 |
|----------|------|
| `never` | 每次匹配都播放（推荐用于音效） |
| `perStage` | 每个阶段只播一次（推荐用于阶段图片） |
| `session` | 整个会话只播一次 |
| `story` | 整个故事只播一次（跨会话） |

## 完整示例

### 示例 1：技能音效

乔峰使用降龙十八掌时播放掌风音效：

```json
{
  "qiaofeng_kanglong": {
    "name": "降龙十八掌",
    "renderer": "audio",
    "durationMs": 3000,
    "trigger": {
      "type": "knowledgeUse",
      "characterId": "qiaofeng",
      "keywords": ["降龙十八掌", "亢龙有悔", "掌力如山"],
      "matchBoldOnly": false
    },
    "playOnce": "never",
    "video": {"containsAudio": false},
    "layers": {},
    "audio": {
      "variants": [
        "assets/performances/wuxia_sfx/audio/zhang_fa_nei_li_bao.wav",
        "assets/performances/wuxia_sfx/audio/zhang_fa_feng_ya.wav"
      ]
    }
  }
}
```

`variants` 数组中有多个文件时，每次随机选一个播放。

### 示例 2：阶段图片 + 音效

进入「山道相遇」阶段时显示场景图片并播放环境音：

```json
{
  "stage_encounter_image": {
    "name": "山道相遇",
    "renderer": "image",
    "durationMs": 6000,
    "trigger": {
      "type": "stageEnter",
      "stageId": "stage_2"
    },
    "playOnce": "perStage",
    "video": {"containsAudio": false},
    "layers": {
      "main": "assets/performances/stage_images/encounter.jpg"
    },
    "audio": {
      "variants": ["assets/performances/wuxia_sfx/audio/wind_mountain.wav"]
    }
  }
}
```

图片演出会同时显示图片和播放音效。

### 示例 3：多层动画技能

```json
{
  "qiaofeng_kanglong_visual": {
    "name": "降龙十八掌·视觉",
    "renderer": "layeredCss",
    "durationMs": 3800,
    "trigger": {
      "type": "knowledgeUse",
      "characterId": "qiaofeng",
      "keywords": ["降龙十八掌"],
      "matchBoldOnly": true
    },
    "playOnce": "session",
    "video": {"containsAudio": false},
    "layers": {
      "bg": "assets/performances/kanglong/images/backgrounds/dark-bg.jpg",
      "characterAttack": "assets/performances/kanglong/images/characters/qiaofeng-attack.png",
      "shockwave": "assets/performances/kanglong/images/skills/shockwave.png",
      "title": "assets/performances/kanglong/images/skills/title.png"
    },
    "audio": {
      "main": "assets/performances/kanglong/audio/full-sequence.mp3"
    }
  }
}
```

## 文件目录结构

```
apps/data/task-packages/虚竹/
├── manifest.json              ← 演出配置在这里
└── assets/performances/
    ├── wuxia_sfx/audio/       ← 通用武侠音效
    │   ├── zhang_fa_feng_ya.wav
    │   └── quan_jiao_zhong_quan.wav
    ├── kanglong/              ← 降龙十八掌专用
    │   ├── audio/
    │   └── images/
    └── stage_images/          ← 阶段场景图片
        ├── encounter.jpg
        └── climax.jpg
```

## 在管理面板配置

1. 进入管理后台 → 选择故事包 → 「演出」标签
2. 点击「新建演出」
3. 填写名称、选择渲染器和触发类型
4. 上传音频/图片资源
5. 配置关键词和播放策略
6. 点击「保存演出」

## 注意事项

- 音频格式支持：mp3、wav、ogg
- 图片格式支持：jpg、png、webp
- `durationMs` 控制演出显示时长（毫秒），音频演出建议设为音频实际时长
- 关键词匹配不区分大小写
- `matchBoldOnly: false` 会匹配全文，可能误触发较多；建议对通用词设为 `true`
- 多个演出可以同时触发（队列播放）
- 用户可通过菜单开关控制演出和闪字显示
