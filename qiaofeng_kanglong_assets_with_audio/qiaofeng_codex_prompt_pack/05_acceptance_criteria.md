# 验收标准

Codex 实现完成后，请按以下标准检查。

## 基础功能

- [ ] 点击按钮或触发事件后，演出层能出现。
- [ ] 乔峰图片能显示。
- [ ] “亢龙有悔”文字或招式图能显示。
- [ ] 掌风、冲击波、闪光、烟尘至少有 3 种视觉效果。
- [ ] 音频能在用户交互后播放。
- [ ] 3.5 到 4.2 秒后自动关闭。
- [ ] 点击“跳过”能立即关闭。
- [ ] 关闭后 DOM、timer、audio 都被清理。

## 视觉体验

- [ ] 背景有暗化和舞台感。
- [ ] 角色有入场或推近动画。
- [ ] 掌风有方向性，从乔峰侧向外轰出。
- [ ] 冲击波有扩散感。
- [ ] 屏幕有轻微震动，但不刺眼。
- [ ] 手机屏幕下不会严重溢出或遮挡按钮。

## 代码质量

- [ ] 素材路径集中配置。
- [ ] 事件 ID 可扩展。
- [ ] 没有硬编码散落在多个文件中。
- [ ] 没有引入不必要的大型依赖。
- [ ] TypeScript 项目中类型完整。
- [ ] 没有 SSR 运行时错误。
