# 乔峰「亢龙有悔」互动演出音效包

包含 WAV 与 MP3 两种格式，适合 Web 页面互动读书演出。

## 文件说明

- wind-rush：掌风蓄力/人物入场呼啸声
- skill-title-flash：招式标题「亢龙有悔」出现时的闪光音
- kanglong-roar：龙吟/内力爆发声
- palm-hit：出掌瞬间冲击声
- screen-impact：震屏爆裂/命中声
- kanglong-full-sequence：已混好的完整 3.6 秒演出音轨

## 推荐触发时间轴

0.15s wind-rush
0.95s skill-title-flash
1.05s kanglong-roar
1.55s palm-hit
1.72s screen-impact

如果只想快速接入网页，直接播放 kanglong-full-sequence.mp3 即可。
