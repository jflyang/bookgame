# Voice Reference Audio Files

Place reference audio files here for each character voice.

## Naming Convention

```
{character_id}_ref.wav    — Primary reference audio
{character_id}_ref.mp3    — Alternative format
```

## Requirements

- Duration: 3-10 seconds of clear speech
- Format: WAV (16kHz or higher), MP3, FLAC, OGG
- Content: A representative sample of the character's voice
- Quality: Clean audio, minimal background noise

## Example Files Needed

```
qiaofeng_ref.wav      — 乔峰：低沉有力的男声
xuzhu_ref.wav         — 虚竹：温和谦逊的年轻僧人
duanyu_ref.wav        — 段誉：温文尔雅的书生
dingchunqiu_ref.wav   — 丁春秋：阴沉狡诈的老年男声
```

## Tips

- Use voice samples from audiobooks, drama CDs, or game voice acting
- Ensure the sample captures the character's distinctive vocal qualities
- Avoid samples with music or sound effects in the background
- CosyVoice works best with 5-8 seconds of reference audio
